# Круг обязанностей SEO-специалиста (специалиста по продвижению сайта)

## 1. Стратегия и аналитика

### 1.1 Анализ бизнеса и ниши
- Понимание бизнес-модели клиента
- Определение целевой аудитории
- Анализ конкурентов (видимость, структура, контент, ссылки)
- Определение KPI (трафик, лиды, продажи, позиции)

### 1.2 Формирование SEO-стратегии
- Определение приоритетных направлений продвижения
- Выбор типов страниц (блог, категории, лендинги)
- Определение региональности и географии
- Планирование этапов продвижения

---

## 2. Работа с семантикой и интентом

### 2.1 Сбор семантического ядра
- Сбор ключевых слов
- Кластеризация запросов
- Определение частотности
- Удаление нерелевантных ключей

### 2.2 Определение поискового интента
- Анализ SERP
- Классификация запросов (informational, commercial, transactional, navigational)
- Привязка ключей к типам страниц

### 2.3 Карта сайта по семантике
- Распределение кластеров по страницам
- Создание структуры разделов
- Избежание каннибализации запросов

---

## 3. Техническая SEO-оптимизация

### 3.1 Технический аудит
- Проверка индексации
- Анализ robots.txt
- Проверка sitemap.xml
- Проверка канонических ссылок
- Поиск дублей страниц
- Проверка редиректов
- Анализ структуры URL

### 3.2 Производительность
- Скорость загрузки
- Core Web Vitals
- Оптимизация изображений
- Lazy loading

### 3.3 Мобильная оптимизация
- Адаптивность
- Mobile-first индексирование

### 3.4 Структурированные данные
- Schema.org разметка
- FAQ schema
- Product schema
- Article schema

---

## 4. On-Page SEO

### 4.1 Оптимизация страниц
- Title
- Meta description
- H1–H6
- Alt-теги
- Внутренние ссылки
- Анкор-листы

### 4.2 Контентная оптимизация
- Создание ТЗ для копирайтеров
- Оптимизация плотности ключей
- Улучшение структуры текста
- Добавление таблиц и списков
- Добавление FAQ-блоков

### 4.3 UX и поведенческие факторы
- Улучшение структуры страницы
- Оптимизация первого экрана
- Улучшение читаемости
- Оптимизация CTA

---

## 5. Контент-стратегия

### 5.1 Планирование контента
- Контент-план
- Структура блоговых разделов
- Планирование evergreen-контента

### 5.2 Управление созданием контента
- Постановка задач авторам
- Редактирование
- Проверка на уникальность
- Проверка на соответствие интенту

### 5.3 Обновление старого контента
- Актуализация информации
- Расширение статьи
- Улучшение структуры

---

## 6. Off-Page SEO (внешнее продвижение)

### 6.1 Линкбилдинг
- Анализ ссылочного профиля
- Поиск доноров
- Outreach
- Размещение гостевых статей

### 6.2 Репутационные сигналы
- Работа с отзывами
- Мониторинг упоминаний
- PR-активность

### 6.3 Контроль качества ссылок
- Проверка токсичных ссылок
- Disavow при необходимости

---

## 7. Локальное SEO (если применимо)

- Настройка профиля в Google Business
- Оптимизация NAP (Name, Address, Phone)
- Работа с локальными каталогами
- Сбор отзывов

---

## 8. Мониторинг и отчетность

### 8.1 Отслеживание позиций
- Мониторинг ключевых запросов
- Анализ динамики

### 8.2 Аналитика трафика
- Источники трафика
- Поведенческие метрики
- Конверсии

### 8.3 Отчеты
- Ежемесячные отчеты
- Анализ роста
- Рекомендации по улучшению

---

## 9. Работа с командой

- Взаимодействие с разработчиками
- Работа с дизайнерами
- Работа с копирайтерами
- Постановка задач в таск-трекере
- Контроль внедрения изменений

---

## 10. Постоянная оптимизация

SEO — это не разовая задача.

Специалист должен:
- Отслеживать изменения алгоритмов
- Проводить регулярные аудиты
- Тестировать гипотезы
- Улучшать конверсию
- Масштабировать успешные страницы

---

# KPI SEO-специалиста

- Рост органического трафика
- Рост видимости
- Рост количества ключей в ТОП
- Рост конверсий из органики
- Улучшение поведенческих метрик

---

# Итог

SEO-специалист отвечает за:

1. Привлечение органического трафика  
2. Рост видимости сайта  
3. Соответствие сайта поисковому интенту  
4. Улучшение технического состояния сайта  
5. Повышение конверсии из органики  

SEO — это стратегическая, техническая и маркетинговая работа одновременно.
