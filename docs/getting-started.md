# Getting Started

Добро пожаловать в SEO MCP Agent! Этот раздел поможет вам быстро начать работу с агентом.

## Содержание

- [Installation](installation.md) - Установка и настройка окружения
- [Quick Start](quick-start.md) - Первые шаги и базовый пример
- [Configuration](configuration.md) - Настройка конфигурации и переменных окружения

## Системные требования

- **Python:** 3.12 или выше
- **OS:** macOS, Linux, Windows
- **RAM:** Минимум 4GB (рекомендуется 8GB для ML-моделей)
- **Disk:** ~2GB для зависимостей и моделей

## Основные компоненты

### Web Interface
FastAPI-based веб-интерфейс на порту 8030.

### Desktop Application
PySide6 десктоп приложение для локального анализа.

### CLI Tool
Командная строка для автоматизации и скриптов.

## Поддерживаемые источники

- **HTTP/HTTPS** сайты через httpx
- **JavaScript-heavy** сайты через PlayWright
- **Локальные HTML** файлы
